    The Design Process

    In the Master Mobile UX video, assigned in Week 1 of this WDD330 course, we learned about designing mobile-first. At about the same time, at work, we had built an app for Cisco Systems and they asked us to make it "mobile-friendly." I found first-hand that it is much easier to design mobile-first than to take an app designed for tablet or Web and make it mobile-friendly.

    Therefore, as I began to write proof-of-concept UI code for the Poet's Pal, I did so using the device tools in Chrome.
