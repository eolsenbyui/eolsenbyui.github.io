Vec Class

The Vec class encapsulates the head of a vector at coordinates x and y. The implicit tail of the vector is the origin coordinate (0, 0).

Constructors

Vec()

Creates an empty Vec object with x and y properties undefined.

Vec(x, y)

Creates a Vec object that initializes the x and y properties with the values passed to the constructor.

Properties

x

The x coordinate of the head of the vector.

y

The y coordinate of the head of the vector.

length

The magnitude of the vector.

angle

The direction of the vector in radians.

degrees

The direction of the vector in degrees.

Methods

plus(Vec)

Add the x and y values of the passed in Vec object to the x and y values, respectively, of this Vec object.

minus(Vec)

Subtract the x and y values of the passed in Vec object from the x and y values, respectively, of this Vec object.
