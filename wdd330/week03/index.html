Notes

  1. Arrow functions have no this pointer.
  2. event.stopPropagation() is kind of like event.preventDefault().
  3. The childNodes property of an element is a NodeList type, not an array. We can make an array of it by using the Array.from() method if we want to use Array.map(), Array.filter(), etc. We can also iterate over it with NodeList.forEach() and for (let child of node.childNodes).
  4. Element nodes represent HTML tags. They are type 1 nodes and the JavaScript constant is Node.ELEMENT_NODE.
  5. Text nodes represent text in the HTML document. They are type 3 nodes. The JavaScript constant is Node.TEXT_NODE.
  6. Comments are type 8 nodes, Node.COMMENT_NODE.
